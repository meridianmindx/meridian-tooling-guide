# Agent Deploy (MVP) Integration

Quick-start to deploying agents end-to-end with MCP and CrewAI primitives.

## Prerequisites

- Python 3.11+
- `pip` or Poetry
- NVIDIA NIM API key (for local LLM endpoints)



## Steps

- [ ] 1. Install `mcp-deploy` package (local wheel OK)
- [ ] 2. Configure MCP runtime with NIM endpoints
- [ ] 3. Run sample crew config via CLI

## Expected

- 1-click multi-agent orchestrator generation
- Zero-config LLMs via NIM
- Local-first telemetry and artifact output
